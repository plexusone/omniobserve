# Release Notes v0.7.0

**Release Date:** March 1, 2026

## Highlights

- Organization rename from agentplexus to plexusone

## Breaking Changes

### Module Path Changed

The Go module path has changed from `github.com/agentplexus/omniobserve` to `github.com/plexusone/omniobserve`.

**Before:**

```go
import "github.com/agentplexus/omniobserve/llmops"
```

**After:**

```go
import "github.com/plexusone/omniobserve/llmops"
```

### Upgrade Guide

Update all import statements in your code:

```bash
# Using sed (macOS)
find . -name "*.go" -exec sed -i '' 's|github.com/agentplexus/omniobserve|github.com/plexusone/omniobserve|g' {} +

# Using sed (Linux)
find . -name "*.go" -exec sed -i 's|github.com/agentplexus/omniobserve|github.com/plexusone/omniobserve|g' {} +
```

Then update your dependencies:

```bash
go get github.com/plexusone/omniobserve@v0.7.0
go mod tidy
```
